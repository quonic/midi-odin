# MIDI Odin

This is a library that lets you use a MIDI grid controller as an LED matrix.

## Supported Devices

1. LaunchPad Mini Mk3

## Considering Supporting Devices

1. All other LaunchPad's
1. [LinnStrument](https://www.rogerlinndesign.com/linnstrument) - [MIDI Document](https://github.com/rogerlinndesign/linnstrument-firmware/blob/master/midi.txt)

## Unsupported Devices

Devices that don't publish a MIDI implimentation document will not be supported.

## Contributions

Clone this repository, make your changes, submit a PR.
